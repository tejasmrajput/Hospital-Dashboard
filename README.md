# 🏥 Hospital Emergency Room Dashboard

## 📌 Overview
An interactive Power BI dashboard that analyzes hospital emergency room data, including patient visits, referrals, demographics, patient satisfaction, and peak arrival times.

## 📊 Key Features
- Total Patients
- Referred Patients
- Male vs Female Ratio
- Race-wise Analysis
- Patient Arrival by Time
- Patient Satisfaction Score
- Date & Year Filters

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Excel
